# Ans)  

**數據摘要 (Data Summary)**：
* **Treatment 1**: 3, 4, 2, 3, 2 ($n_1=5, T_1=14$)
* **Treatment 2**: 2, 3, 5, 2, 5 ($n_2=5, T_2=17$)
* **Treatment 3**: 2, 0, 2, 1 ($n_3=4, T_3=5$)
* **Overall**: $n=14$, $G=36$

---
#### **1. 設立假設**
* $H_0: \mu_1 = \mu_2 = \mu_3$
* $H_1$: At least one mean is different

#### **2. 決定顯著水準**
* $\alpha = 0.05$

#### **3. 檢定統計量與分佈**
$$F_{STAT} = \frac{MST}{MSE} \sim F_{(k-1), (n-k)}$$

* 在此題中：
    * $df_1 = k - 1 = 3 - 1 = 2$
    * $df_2 = n - k = 14 - 3 = 11$
    * 所以檢定統計量服從 **$F_{2, 11}$** 分佈。

#### **4. 計算統計值**

**(1) 計算變異數分析所需數值**
* **CM**:
    $$CM = \frac{G^2}{n} = \frac{36^2}{14} = 92.5714$$
* **Total SS**:
    $$Total\ SS = \sum x_{ij}^2 - CM = 118 - 92.5714 = \mathbf{25.4286}$$
* **SST**:
    $$SST = \sum \frac{T_i^2}{n_i} - CM = \left(\frac{14^2}{5} + \frac{17^2}{5} + \frac{5^2}{4}\right) - 92.5714$$
    $$= (39.2 + 57.8 + 6.25) - 92.5714 = \mathbf{10.6786}$$
* **SSE**:
    $$SSE = Total\ SS - SST = 25.4286 - 10.6786 = \mathbf{14.7500}$$

**(2) 填寫變異數分析表 (ANOVA Table)**

| Source | df | SS | MS | F |
| :--- | :--- | :--- | :--- | :--- |
| **Treatments** | $2$ | $10.6786$ | $MST = 5.3393$ | $F_{STAT} = \mathbf{3.98}$ |
| **Error** | $11$ | $14.7500$ | $MSE = 1.3409$ | |
| **Total** | $13$ | $25.4286$ | | |

#### **5. 拒絕域與 P 值**

* **Rejection Region (拒絕域)**:
    查 F 表 ($df_1=2, df_2=11, \alpha=0.05$)，臨界值 $F_{0.05} = 3.98$。
    **Reject $H_0$ if $F_{STAT} > 3.98$**

* **P-value**:
    計算出的 $F_{STAT} \approx 3.982$ 略大於臨界值 3.98。
    因此 **P-value $< 0.05$** (但非常接近 0.05)。

#### **6. 結論**

* **Decision**: 因為 $F_{STAT} = 3.982 > 3.98$，落入拒絕域 $\rightarrow$ **Reject $H_0$**。
* **Conclusion**: 在 $\alpha = 0.05$ 下，有足夠證據顯示這三組的平均數有顯著差異。
