### 📝 題目翻譯

**15. 木材的成本 (The Cost of Lumber)**
一位建商想要比較他在四個計畫開始施工的州，其標準或更高等級框架木材每 1000 板英尺 (board meters) 的價格。他在每個州隨機選擇了 5 家供應商。價格如表所示（單位：美元）。

**問題：**
* **a.** 使用了哪種類型的實驗設計？
* **b.** 為這些數據建構變異數分析 (ANOVA) 表。
* **c.** 數據是否提供足夠的證據顯示，這四個州的框架木材平均價格有所不同？使用顯著水準 $\alpha = .05$ 進行檢定。

---


#### **Ans.a)**

完全隨機設計 (Completely Randomized Design, CRD)

---

#### **Ans.b)**

**Step 1: 數據整理**
* $k = 4$ (組數)
* $n_i = 5$ (每組樣本數)
* $n = 20$ (總樣本數)
* 各組總和 ($T_i$):
    * $T_1 = 1311$
    * $T_2 = 1174$
    * $T_3 = 1258$
    * $T_4 = 1343$
* 總和 ($G$): $5086$

**Step 2: 計算校正因子 (CF)**  
$$CF = \frac{G^2}{n} = \frac{(5086)^2}{20} = \mathbf{1,293,369.8}$$

**Step 3: 計算平方和 (Sum of Squares)**

1.  **Total SS (總平方和):**  
    $$Total\ SS = \sum x^2 - CF = 1,297,302 - 1,293,369.8 = \mathbf{3,932.2}$$

2.  **SST (處理平方和, Treatment SS):**  
    $$SST = \sum \frac{T_i^2}{n_i} - CF$$  
    $$SST = \left( \frac{1311^2}{5} + \frac{1174^2}{5} + \frac{1258^2}{5} + \frac{1343^2}{5} \right) - 1,293,369.8$$  
    $$SST = 1,296,642 - 1,293,369.8 = \mathbf{3,272.2}$$  

3.  **SSE (誤差平方和, Error SS):**  
    $$SSE = Total\ SS - SST = 3,932.2 - 3,272.2 = \mathbf{660.0}$$

**Step 4: 計算均方 (MS) 與 F 值**  
* $MST = \frac{3272.2}{3} = \mathbf{1090.73}$  
* $MSE = \frac{660.0}{16} = \mathbf{41.25}$  
* $F = \frac{1090.73}{41.25} = \mathbf{26.44}$  

**【ANOVA Table】**

| Source of Variation | Sum of Squares (SS) | Degrees of Freedom (df) | Mean Square (MS) | F |
| :--- | :---: | :---: | :---: | :---: |
| **Treatments** | **SST** = 3,272.2 | 3 | **MST** = 1,090.73 | **26.44** |
| **Error** | **SSE** = 660.0 | 16 | **MSE** = 41.25 | |
| **Total** | **Total SS** = 3,932.2 | 19 | | |

#### **c. Hypothesis Test (假設檢定)**

**1. 設立虛無假設與對立假設 (Hypotheses)**
* $H_0: \mu_1 = \mu_2 = \mu_3 = \mu_4$
* $H_1:$ Not all $\mu_j$ are equal

**2. 決定顯著水準 (Level of Significance)**
* $\alpha = 0.05$

**3. 檢定統計量與分佈 (Test Statistic)**  
$$F_{STAT} = \frac{MST}{MSE} \sim F_{3, 16}$$  

**4. 設立決策法則 (Decision Rule)**  
* Critical Value: $F_{0.05}(3, 16) = 3.24$  
* Rejection Rule: If $F_{STAT} > 3.24$, reject $H_0$.  

**5. 計算檢定統計量 (Compute Test Statistic)**  
$$F_{STAT} = \frac{1090.73}{41.25} = \mathbf{26.44}$$  

**6. 統計決策與結論 (Decision & Conclusion)**
* **Decision:** 因為 $26.44 > 3.24$, **Reject $H_0$**.
* **Conclusion:** 有充分的證據表明，四個州之間，框架用木材每千板米的平均價格存在差異
