#### **a. 進行變異數分析 (ANOVA)**

**前置計算 (Preliminary Calculations)**

* **方法 A ($i=1$):** $n_1 = 5$, $T_1 = 380$
* **方法 B ($i=2$):** $n_2 = 3$, $T_2 = 199$
* **方法 C ($i=3$):** $n_3 = 3$, $T_3 = 261$
* **總體 ($Total$):**
    * 總樣本數 $N = 11$
    * 總和 $G = 380 + 199 + 261 = 840$
    * 總平方和 $\sum x_{ij}^2 = 29018 + 13433 + 22835 = 65286$

**平方和計算 (Sum of Squares Calculations)**

* **校正因子 (Correction Factor, CF):**
    $$CF = \frac{G^2}{N} = \frac{840^2}{11} \approx \mathbf{64145.4545}$$

* **總平方和 (Total SS):**
    $$SS_{Total} = \sum x_{ij}^2 - CF = 65286 - 64145.4545 = \mathbf{1140.5455}$$

* **處理平方和 (Treatment SS, SST):**
    $$SST = \sum \frac{T_i^2}{n_i} - CF = \left( \frac{380^2}{5} + \frac{199^2}{3} + \frac{261^2}{3} \right) - 64145.4545$$
    $$SST = (28880 + 13200.3333 + 22707) - 64145.4545$$
    $$SST = 64787.3333 - 64145.4545 = \mathbf{641.8788}$$

* **誤差平方和 (Error SS, SSE):**
    $$SSE = SS_{Total} - SST = 1140.5455 - 641.8788 = \mathbf{498.6667}$$

**均方 (MS) 與 F 值計算**

* $MST = \frac{641.8788}{2} = \mathbf{320.9394}$
* $MSE = \frac{498.6667}{8} = \mathbf{62.3333}$
* $F = \frac{320.9394}{62.3333} \approx \mathbf{5.15}$

**變異數分析表 (ANOVA Table)**

| 變異來源 (Source) | 自由度 (df) | 平方和 (SS) | 均方 (MS) | F 檢定統計量 |
| :--- | :---: | :---: | :---: | :---: |
| **處理 (Treatments)** | 2 | 641.8788 | 320.9394 | **5.15** |
| **誤差 (Error)** | 8 | 498.6667 | 62.3333 | |
| **總和 (Total)** | 10 | 1140.5455 | | |

#### **b. 假設檢定 (Hypothesis Test) - SOP 6 步驟**

**Step 1: 設立假設 (Set up Hypotheses)**
* $H_0: \mu_A = \mu_B = \mu_C$ (三種方法的平均分數皆相同)
* $H_1:$ 至少有兩組平均分數不相等 (At least two means differ)

**Step 2: 決定顯著水準 (Select Level of Significance)**
* $\alpha = 0.05$ (未指定則預設 0.05)

**Step 3: 檢定統計量與分佈 (Determine Test Statistic and Distribution)**

$$F_{STAT} = \frac{MST}{MSE} \sim F_{k-1, N-k}$$
$$F_{STAT} \sim F_{2, 8}$$

**Step 4: 設立決策法則 (Establish Decision Rule)**
* 查表：$F_{0.05}(2, 8)$ * 臨界值 (Critical Value) = **4.46**
* **拒絕域 (Rejection Region):** 若 $F_{STAT} > 4.46$，則拒絕 $H_0$。

**Step 5: 計算檢定統計量 (Compute Test Statistic)**

$$F_{STAT} = \mathbf{5.15}$$

**Step 6: 統計決策與結論 (Decision and Conclusion)**
* **決策:** 因為 $5.15 > 4.46$，落在拒絕域內，故 **拒絕 $H_0$**。
* **結論:** 有充分證據顯示三種方法處理後的學生平均分數存在顯著差異。
