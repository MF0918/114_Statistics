**前置計算 (基礎統計量)**

* 令 臂展 ($x$) 為自變數。
* 令 身高 ($y$) 為應變數。
* 樣本數 $n = 8$。

**數據摘要表：**

| 統計量 | 數值 |
| :--- | :--- |
| $\sum x$ (x 總和) | $1328$ |
| $\sum y$ (y 總和) | $1331$ |
| $\sum x^2$ (x 平方和) | $221,016$ |
| $\sum y^2$ (y 平方和) | $221,877$ |
| $\sum xy$ (交叉乘積和) | $221,414$ |
| $\bar{x}$ (x 平均) | $166$ |
| $\bar{y}$ (y 平均) | $166.375$ |

**離差平方和公式：**

$$S_{xx} = \sum x^2 - \frac{(\sum x)^2}{n} = 221,016 - \frac{(1328)^2}{8} = 221,016 - 220,448 = \mathbf{568}$$

$$S_{xy} = \sum xy - \frac{(\sum x)(\sum y)}{n} = 221,414 - \frac{(1328)(1331)}{8} = 221,414 - 220,946 = \mathbf{468}$$

**描述：**
散佈圖顯示兩變數之間呈現 **強烈的正向線性關係**。隨著臂展增加，身高也有增加的趨勢，且數據點緊密地圍繞在一條直線周圍。

---

#### **Ans.b)**

如果臂展大致等於身高，則數學關係式為 $y = x$。
因此，理論上的迴歸線斜率應為 **1**。

---

#### **Ans.c)**  

**1. 計算斜率 ($b_1$)：**

$$b_1 = \frac{S_{xy}}{S_{xx}} = \frac{468}{568} \approx \mathbf{0.8239}$$

**2. 計算截距 ($b_0$)：**

$$b_0 = \bar{y} - b_1\bar{x} = 166.375 - (0.8239)(166)$$
$$b_0 = 166.375 - 136.7674 = \mathbf{29.6076}$$

**3. 迴歸線方程式：**

$$\hat{y} = b_0 + b_1x$$
$$\hat{y} = 29.61 + 0.82x$$

**4. 關於斜率的結論：**
計算出的斜率 $b_1 \approx 0.82$。
雖然這證實了兩者有強烈的正相關，但數值 **0.82** 與達文西假設的完美 **1** 略有差距。這顯示在這個樣本中，身高的增長幅度略小於臂展的增長幅度。

---

#### **Ans.d)** 

$$\hat{y} = 29.6076 + 0.8239(157)$$
$$\hat{y} = 29.6076 + 129.3523$$
$$\hat{y} \approx \mathbf{158.96} \text{ 公分}$$

如果一個人的臂展為 157 公分，我們預測其身高約為 **159 公分** (或 158.96 公分)。
