# Ans:  

$p=0.47$  
$n=100$  

-----

## a.

要使用常態分佈來近似樣本比例分佈，必須滿足 $np \ge 5$ 和 $n(1-p) \ge 5$ 的條件。

$np = 100 \times 0.47 = 47$ （大於 $5$）  
$n(1-p) = 100 \times (1 - 0.47) = 100 \times 0.53 = 53$ （大於 $5$）    

由於兩個條件都滿足，因此樣本比例 $\hat{p}$ 的分佈近似於常態分佈。 

$\mu_{\hat{p}} = p = 0.47$  
$\sigma_{\hat{p}} = \sqrt{\frac{p(1-p)}{n}} = \sqrt{\frac{0.47 \times 0.53}{100}} \approx 0.0499$

## b.  

計算 $P(\hat{p} < 0.45)$

Z轉換：  
$P(\hat{p} < 0.45)= P( \frac{\hat{p} - p}{\sigma_{\hat{p}}} < \frac{0.45 - 0.47}{0.0499} ) = P(Z<-0.4)$  

查表:  
$P(Z<-0.4):0.3443$  

$P =$ `0.3443`  

## c.

計算 $P(0.42 < \hat{p} < 0.45)$

Z轉換：  
$P(0.42 <\hat{p} < 0.45)= P( \frac{0.42 - 0.47}{0.0499} < \frac{\hat{p} - p}{\sigma_{\hat{p}}} < \frac{0.45 - 0.47}{0.0499} ) = P(-1 < Z < -0.4)=P(Z < -0.4) - P(Z < -1)$  

查表:  
$P(Z < -0.4): 0.3446$  
$P(Z < -1): 0.1587$  

$P=0.3446 - 0.1587 =$ `0.1859`  

## d.

計算 Z 分數：  
$$Z = \frac{0.30 - 0.47}{0.0499} \approx \frac{-0.17}{0.0499} \approx -3.41$$

查表:   
$P(Z<-0.3.41): 0.0003$   

$P=$ `0.0003` 

由於發生此事件的機率極低 ($P \approx 0.0003$),強烈懷疑 $47\%$ 這個數字是錯誤的。
