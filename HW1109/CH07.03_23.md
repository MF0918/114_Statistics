# Ans:
好的，我們來詳細解答這個關於電池壽命的機率問題。

這個問題的核心是應用**中央極限定理 (Central Limit Theorem, CLT)** 來處理樣本平均數的抽樣分配。即使我們不知道單一電池壽命的母體分佈是什麼樣子，只要樣本數夠大，樣本平均數的分佈就會近似於常態分佈。

### **第一步：建立樣本平均數的抽樣分配模型**

首先，我們整理題目給的資訊：

  $\mu=1110$  
  $\sigma=80$   
  $n=400$  

* 因為樣本數 $n = 400$ 遠大於經驗法則的 30，根據中央極限定理，樣本平均壽命 ($\bar{x}$) 的抽樣分配會近似於常態分佈。

* 抽樣分配的平均值等於母體平均值。  

  $\mu_{\bar{x}} = \mu = 1110$  

* $\sigma_{\bar{x}} = \frac{\sigma}{\sqrt{n}} = \frac{80}{\sqrt{400}} = \frac{80}{20} = 4 $  

記作:   

$$\bar{x} \sim N(\mu=1110, \sigma=4)$$  

標準化成Z-score的公式是：  

$$ Z = \frac{\bar{x} - \mu_{\bar{x}}}{\sigma_{\bar{x}}} $$  

**a.**  
計算 $P(1100 < \bar{x} < 1110)$

Z標準化轉換：  
$P ( \frac{1100 - \mu_{\bar{x}}}{\sigma_{\bar{x}}} < \frac{\bar{x} - \mu_{\bar{x}}}{\sigma_{\bar{x}}} < \frac{1110 - \mu_{\bar{x}}}{\sigma_{\bar{x}}} )$
$=P ( \frac{1100 - 1110}{4} < \frac{\bar{x} - \mu_{\bar{x}}}{\sigma_{\bar{x}}} < \frac{1110 - 1110}{4})$
$=P ( -2.5 < Z < 0 )$
$=P ( Z < 0 ) - P ( Z < -2.5 )$   

查表:  
$P(Z<0)$ : 0.5000  
$P(Z<-2.5)$ : 0.0062  

$P = 0.5000 - 0.0062 =$ `0.4938`  


**b.**  
計算 $P(\bar{x} > 1120)$

Z標準化轉換：  
$P (  \frac{\bar{x} - \mu_{\bar{x}}}{\sigma_{\bar{x}}} > \frac{1120 - \mu_{\bar{x}}}{\sigma_{\bar{x}}} )$
$=P (  \frac{\bar{x} - \mu_{\bar{x}}}{\sigma_{\bar{x}}} > \frac{1120 - 1110}{4} )$
$=P (Z > 2.5 )$
$= 1 - P ( Z < 2.5 )$   

查表:  
$P(Z>2.5)$ : 0.9938 

$P = 1 - 0.9938 =$ `0.0062`  



**c.**  
計算 $P(\bar{x} < 900)$

Z標準化轉換：  
$P (  \frac{\bar{x} - \mu_{\bar{x}}}{\sigma_{\bar{x}}} < \frac{900 - \mu_{\bar{x}}}{\sigma_{\bar{x}}} )$
$=P (  \frac{\bar{x} - \mu_{\bar{x}}}{\sigma_{\bar{x}}} < \frac{900 - 1110}{4} )$
$=P (Z < -52.5 )$

查表:  
$P(Z<-52.5)$ : 在表外，應趨近於0  

$P \approx$ `0`
